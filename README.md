## 项目简介

Expandable Toolbox 是一个功能强大的可扩展工具箱，旨在为个人和组织提供一个统一、美观且高效的工具管理入口。它可以帮助您轻松组织各种本地可执行文件（`.exe`）、脚本（`.cmd`, `.bat`）以及其他常用工具。

## 核心特性

- **高效异步扫描**：基于 `QtConcurrent` 的多线程扫描机制，即便工具文件夹庞大，也能保持界面响应丝滑。
- **现代化 UI 设计**：
  - **精美弹窗**：重构了“关于”和“系统资源”弹窗，采用 Google Material Design 风格的扁平化卡片布局。
  - **视图模式**：支持“列表”、“图标”和“详情”三种视图，满足不同展示需求。
  - **流畅交互**：优化了窗口渲染逻辑，解决了移动卡顿与闪烁问题。
-  全方位的工具管理**：
  - **增删改查**：支持直接从界面添加、删除工具，以及重命名工具和编辑描述。
  - **元数据管理**：通过统一的 `metadata.json` 自动同步工具配置。
- **高级启动配置**：
  - **提权运行**：支持一键开启“以管理员身份运行”。
  - **自定义参数**：支持为每个工具单独配置命令行参数。
- **深度系统集成**：
  - **最小化至托盘**：关闭按钮直接退出，最小化按钮缩至托盘，保证后台静默运行。
  - **实时资源监控**：状态栏实时查看 CPU、内存、硬盘及 GPU 占用情况，数据采集完全异步化，不占用主线程。
- **智能搜索**：毫秒级实时搜索，支持按名称和描述快速定位。

## 安装与使用

### 安装方法

1. 从 Releases 页面下载最新版本的可执行文件。
2. 或解压下载的 `.7z` 压缩包到您选择的目录。
3. 运行 `Expandable Toolbox.exe` 启动应用程序。

### 使用说明

#### 1. 管理分类与工具
- **分类操作**：左侧面板右键点击可“添加分类”或“重命名分类”。
- **添加工具**：右键点击右侧工具区域，选择“添加工具”。
- **配置工具**：右键点击具体工具卡片，可配置“重命名”、“编辑描述”、“启动参数”或“管理员权限”。

#### 2. 系统监控
- 点击状态栏右侧的 **系统资源** 即可弹出监控窗口。数据每 1.5 秒异步更新一次。

#### 3. 视图切换
- 使用右上角的下拉框切换不同的显示风格（列表、图标、详情）。

## 开发者信息

### 技术栈

- **开发语言**：C++17
- **GUI 框架**：Qt 6.7.2
- **并行计算**：Qt Concurrent & QFuture (用于异步扫描与数据监控)
- **图形处理**：Qt Graphics Effect (用于平滑渲染)
- **构建系统**：CMake 3.16+
- **编译器**：MinGW 64-bit
- **开发环境**：Qt Creator 14.0.0

### 项目结构

```
Expandable-Toolbox/
├── CMakeLists.txt                 # 顶层构建配置
├── resources.qrc                  # Qt 资源文件 (图标、图片)
├── app_icon.rc                    # Windows 应用图标
├── DirectoryArea.cpp/.h           # 左侧分类管理组件
├── FunctionArea.cpp/.h            # 右侧工具管理核心（异步扫描/元数据）
├── mainwindow.cpp/.h/.ui          # 主窗口（UI 布局/托盘/资源监控逻辑）
└── tools/                         # 工具数据存储目录
    └── [分类名]/
        └── metadata.json          # 关键：存储该分类下所有工具的配置
```

## ⚠️ 注意事项

- **图标显示**：若托盘图标不显示，请确保 `resources.qrc` 已正确编译。
- **管理员权限**：部分系统工具需要提权运行，建议右键点击工具配置“管理员权限”。
- **数据同步**：删除工具会同步删除磁盘物理文件，请谨慎操作。
